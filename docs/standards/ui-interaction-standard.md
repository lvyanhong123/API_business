# UI/交互规范 (UI/Interaction Standard)

## 概述

本文档定义**视觉样式规范**。这些规范只影响页面的视觉效果，修改后不影响功能实现。

---

## 菜单样式规范

### 箭头图标

- **只允许使用**简约的 `>` 字符作为箭头图标
- **禁止使用** `▶` 等花哨符号

### 子菜单样式

- 子菜单需要设置 `max-height` 限制高度
- 需要添加 `overflow-y: auto` 实现滚动

### 滚动条样式

- 滚动条需要与背景融合
- 使用半透明白色，如 `rgba(255, 255, 255, 0.3)` 或 `#ffffff30`

### 多菜单组样式

当存在多个展开菜单组时，`toggleMenuGroup` 和 `renderMenu` 函数必须：

1. 通过 `id` 或 `textContent` 正确区分不同菜单组
2. 分别为每个菜单组的子菜单设置 `open` 类
3. 分别为每个菜单组标题设置 `active` 类

---

## 搜索区域样式规范

### 搜索区块样式

- 搜索区块使用 `.card` + `.card-header` + `.search-bar` 结构
- 搜索框使用 `.search-input` 样式
- 按钮使用 `.btn.btn-small` 样式
- 重置按钮使用 `.cancel-btn` 样式

---

## 列表区域样式规范

### 表格样式

- 表格包裹在 `.table-wrap` 容器中
- 表头使用 `<thead>`，表体使用 `<tbody>`
- 搜索结果统计使用 `.search-results` 类

### 分页器样式

- 分页器使用 `.pagination` 类
- 当前页按钮添加 `active` 类
- 禁用状态按钮添加 `disabled` 属性

---

## 通用样式规范

### 按钮样式

| 按钮类型 | 样式类 |
|----------|--------|
| 普通按钮 | `.btn` |
| 小按钮 | `.btn.btn-small` |
| 取消/重置按钮 | `.cancel-btn` |

### 卡片样式

| 卡片部分 | 样式类 |
|----------|--------|
| 卡片容器 | `.card` |
| 卡片头部 | `.card-header` |
| 卡片标题 | `.card-title` |
| 搜索栏 | `.search-bar` |

### 头像菜单样式

- 页面右上角显示蓝色圆形头像
- 点击头像下拉显示退出登录选项
- 点击外部自动收起下拉菜单
