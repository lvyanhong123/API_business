## 一、账户详情页刷新空白问题

### 问题现象
从客户列表点击"账户"按钮可以正常打开账户详情Tab，但刷新页面后该Tab显示空白。

### 排查过程
1. 检查前端`loadCustomerAccount`函数 - 无语法错误
2. 检查Tab状态恢复逻辑 - 逻辑正常
3. 启动服务器时发现启动失败
4. 发现错误：`adminRoutes.js`引用`{ adminAuth }`，但`auth.js`只导出了默认的`auth`

### 根本原因
`src/middleware/auth.js` 缺少 `adminAuth` 命名导出，导致：
- `adminRoutes.js` 加载时得到 `undefined`
- `Route.get() requires a callback function but got a [object Undefined]`
- 服务器启动失败
- 所有API请求都无法正常工作

### 修复方案
```javascript
// auth.js
const adminAuth = auth;
module.exports = auth;
module.exports.adminAuth = adminAuth;
```

同时删除`app.js`中重复的admin-change-requests路由定义。

## 三、技术要点

### 路由注册检查
添加新路由时，确保：
1. 控制器方法已导出
2. 中间件已正确导出
3. app.js中无重复路由定义
