规则5详解：多接口次数赠送计费

📌 规则5：多接口次数赠送计费 - 逻辑说明

  • 主接口：正常计费，按单价收费（如 0.1元/条）
  • 赠送接口：免费额度 = 主接口调用次数（即主接口每调用1次，赠送接口免费1次）
  • 超出部分：赠送接口如果调用超过免费额度，超出部分按单独单价收费（如 0.2元/条）
接口组配置
接口组成员
车辆信息查询 API-001
主接口 ¥0.10/条
车辆违章查询 API-002
赠送接口 超量 ¥0.20/条
高速通行记录 API-003
赠送接口 超量 ¥0.15/条
计费规则
主 主接口调用次数 × 单价 API-001 × ¥0.10
赠 min(赠送接口调用, 主接口调用) 免费 = 免费额度
赠 max(0, 赠送调用 - 免费额度) × 超量单价 = 超量费用
计算示例
假设当月调用情况
主接口（API-001）调用次数1000 次
赠送接口（API-002）调用次数800 次
赠送接口（API-003）调用次数1200 次
计算过程： ① 主接口费用 1000 次 × ¥0.10 = ¥100.00 ② API-002（违章查询）费用 免费额度 = min(800, 1000) = 800 次 超量 = max(0, 800 - 800) = 0 次 费用 = 0 × ¥0.20 = ¥0.00 ③ API-003（高速通行）费用 免费额度 = min(1200, 1000) = 1000 次 超量 = max(0, 1200 - 1000) = 200 次 费用 = 200 × ¥0.15 = ¥30.00 ───────────────────────────── 合计费用：¥100.00 + ¥0.00 + ¥30.00 = ¥130.00
再举一例
主接口（API-001）调用次数500 次
赠送接口（API-002）调用次数600 次
赠送接口（API-003）调用次数300 次
① 主接口费用 500 × ¥0.10 = ¥50.00 ② API-002 费用 免费额度 = min(600, 500) = 500 次 超量 = 100 次 费用 = 100 × ¥0.20 = ¥20.00 ③ API-003 费用 免费额度 = min(300, 500) = 300 次 超量 = 0 次 费用 = ¥0.00 ───────────────────────────── 合计：¥50 + ¥20 + ¥0 = ¥70.00
核心公式
┌─────────────────────────────────────────────────────────────┐ │ 费用 = 主接口次数 × 主接口单价 │ │ + Σ( max(0, 赠送接口调用次数 - 主接口调用次数) × 赠送超量单价 ) │ │ │ │ 即：每个赠送接口的费用 = max(0, 赠送调用 - 主接口调用) × 超量单价 │ └─────────────────────────────────────────────────────────────┘
简化理解： 赠送接口的免费额度 = 主接口调用次数
赠送接口超出免费额度 = max(0, 赠送调用 - 主接口调用)
